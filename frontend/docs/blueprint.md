# **App Name**: Book Haven

## Core Features:

- Authentication: Secure user authentication (login, register) with JWT token storage and role-based access control (guest, reader, admin).
- Browse Books: Display books in a responsive grid layout with cover image, title, author, average rating, and availability, and include search, filter, sort, and pagination functionalities.
- Book Details: Show detailed information about a selected book, including reviews with ratings and comments; enable liking reviews and adding new reviews by logged-in users.
- User Dashboard: Provide personalized dashboards for readers with sections for currently reading, completed books, reading goals, and quick statistics.
- My Profile: Allow users to view and edit their profile information (name, bio, profile image, reading interests) and follower/following counts.
- Wishlist: Enable users to maintain a list of saved books for later, including adding books to the wishlist and removing them.
- Reading Tracker: Allow users to track their reading progress, including marking books as reading or completed and displaying reading history.

## Style Guidelines:

- Primary color: Deep brown (#A68B64) evoking a sense of warmth and comfort.
- Background color: Light beige (#F5F5DC) provides a soft, neutral backdrop.
- Accent color: Burnt orange (#CC6633) to draw attention to calls to action.
- Body and headline font: 'Alegreya', a serif font, will be used to provide an elegant and intellectual feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a consistent set of line icons for navigation and actions, ensuring clarity and elegance.
- Implement a grid-based layout for books and dashboard sections, optimizing responsiveness and visual coherence across devices.
- Incorporate subtle animations (e.g., fade-in effects, transition animations) to enhance user experience and provide visual feedback.
